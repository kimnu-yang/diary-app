## Intro
: &nbsp; 해당 프로젝트는 API 서버를 통해 데이터 백업이 가능한 다이어리 어플리케이션 개발 프로젝트로 자바 스프링부트를 이용해 1차적으로 API 서버를 구성하고 안드로이드, IOS 어플 환경에 맞게 개발하여 출시까지 진행할 프로젝트입니다.

## Project Summary
- 개발언어
  - Java 17
- 프레임워크
  - Spring Boot 3.2.1
- 데이터베이스
  - MySQL 8.0.2
- ORM
  - JPA
- 빌드
  - Gradle / Groovy
- 웹 인증 방식
  - Header: Json Web Token
- API 명세
  - Swagger(Springdoc)
